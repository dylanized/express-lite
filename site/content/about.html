About Peezy

Peezy is a micro-CMS built in Node.js.

Using it is very easy:

  • clone this repo
  • customize the theme in /site/themes/default
  • add content to /site/content/
  • start the server
This is my experiment in building a minimalist flat file publishing engine built around Express.js. Inspired heavily by WordPress, Jekyll and Harp. Not quite functional yet - Work in progress! by @dylanized
